# Doc Site Portfolio Tutorial

Build a professional technical writing portfolio site using MkDocs and AI assistance.

## What You'll Learn

This tutorial guides you through creating a portfolio documentation site that showcases your technical writing work.

## Tutorial Steps

1. [Setup Environment](step-1-setup.md)
2. [Plan Your Site](step-2-plan-site.md)
3. [Generate Structure](step-3-generate-structure.md)
4. [Write Content](step-4-write-content.md)
5. [Refine with AI](step-5-refine-with-ai.md)
6. [Deploy](step-6-deploy.md)

## Time Commitment

This tutorial takes approximately 4-6 hours to complete.

## Prerequisites

- Basic familiarity with Markdown
- GitHub account
- AI provider API key (Anthropic Claude or OpenAI GPT)
- Text editor

## What You'll Build

A professional portfolio site with:
- About page highlighting your experience
- Project showcase with case studies
- Writing samples
- Contact information
